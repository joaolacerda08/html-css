Usando Links Externos

O de uso de links externos nada mais é do que a função de ser redirecionado para algum site, sendo ele o seu própio ou não. Para fazer o uso dessa função, usamos o parâmetro "a".


Exemplo:

Acesse aqui uma informação de um possível gato.

Para redirecionar um link externo porém sem sobrescrever o seu site, utilizamos dentro do parâmetro "a" essa função: target="_blank" rel="external" essa pequena linha vai fazer com que ao clicar na interação que leva a um link externo, também abra uma nova guia em seu navegador.


Exemplo:

Veja aqui um tutorial de como funciona um possível gato.

Para criar um link que faça um download, basta ao invés de inserir um link website no seu parâmetro "a", seja inserido o arquivo o qual será efetuado o download que, por sua vez, deve estar na mesma pasta do website.

Para você criar um link para redirecionar para uma sefunda página do seu site, basta ao invés de inserir um link em: HREF="LINK" você deve inserir o local do arquivo do seu site.
