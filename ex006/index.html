Principais Formatações

Negrito/Destaque

Para deixar uma parte do texto em Negrito, temos duas maneiras, utilizando a tag "B", exemplo de texto em negrito de maneira NÃO SEMÂNTICA, já para usar o mesmo texto em negrito de maneira semântica, utilizamos a tag "strong", ficando assim em negrito ou destaque de maneira semântica.

Itálico/Ênfase

Para deixar uma parte do texto em itálico também temos duas maneiras, sando a primeira delas usando a tag "i" fazendo assim o texto ficar em itálico de maneira não semântica, ou utilzando a tag "em", fazendo assim o texto ficar em itálico de maneira semântica.

Texto Marcado

Além dessas, temos também a tag "mark", responsável por marcar textos, como se fossem marcados por uma caneta marca texto.

Texto Grande e Pequeno

Agora, vamos usar as tags "big" e "small", sendo a tag "big" responsável por deixar as letras levemente maiores e a tag "small" responsável por deixar as letras levemente menores, porem, a tag "big" é considerada como OBSOLETA já a tag small não.

Texto Deletado

Podemos marcar um texto como texto excluido, para indicar que é um texto que deve ser lido mas não considerado, a tag responsável por isso é a tag "del".
