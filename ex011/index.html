Imagens Dinámicas

Agora voce vai ver um exemplo de imagem que muda dependendo do tamanho da tela!

A imprtância de uma imagem dinámica é a adaptatividade do website para computadores, tablets e telefones, sem a imagem ficar "carregada pela metade" para usar essa função, usamos a tag "picture", dentro dela, inserimos um "img" que por sua vez, colocamos primeiro a nossa maior imagem, depois, adicionamos a cima do img, um "source:media" mudamos o valor de min-width para max-width e inserimos um valor, normalmente 1025, para transformar uma imagem grande em média, e 725 para transformar uma imagem média para pequena. Após definir tudo, ainda dentro do parâmetro, teve ser inserido um "type="image.png" para funcionar perfeitamente.
